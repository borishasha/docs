UI 컴포넌트 및 레이아웃 ​

개요 ​

src/views/ui/ 디렉토리는 프로젝트 전반에서 사용되는 공통 UI 컴포넌트와 레이아웃 컴포넌트를 포함합니다.

디렉토리 구조 ​

src/views/ui/
├── components/                # 공통 컴포넌트
│   ├── common/               # 기본 컴포넌트
│   │   ├── button/
│   │   │   └── ButtonBox.vue
│   │   ├── input/
│   │   │   └── InputPhone.vue
│   │   ├── Loading.vue
│   │   ├── lookup/
│   │   │   └── LookUpBox.vue
│   │   ├── select/
│   │   │   ├── MultiSelectBoxSQL.vue
│   │   │   ├── SelectBoxDate.vue
│   │   │   └── SelectBoxSQL.vue
│   │   └── tree/
│   │       ├── TreeDeptDropdown.vue
│   │       ├── TreeEmpDropdown.vue
│   │       └── TreeOrgNode.vue
│   ├── error/                # 에러 컴포넌트
│   │   └── ErrorHeader.vue
│   ├── grid/                 # 그리드 컴포넌트
│   │   ├── GridClickField.vue
│   │   ├── GridContextMenu.vue
│   │   └── GridMultiSelectWrapper.vue
│   ├── main/                 # 메인 컴포넌트
│   │   └── MainHeader.vue
│   ├── menu/                 # 메뉴 컴포넌트
│   │   ├── MenuBmk.vue
│   │   ├── MenuHeaderBar.vue
│   │   └── MenuSideBar.vue
│   └── popup/                # 팝업 컴포넌트
│       ├── PopupApproval.vue
│       ├── PopupApprovalBox.vue
│       ├── PopupApproveList.vue
│       ├── PopupFileLink.vue
│       ├── PopupLog.vue
│       ├── PopupPageLog.vue
│       └── PopupSmsTemplateList.vue
├── layouts/                  # 레이아웃 컴포넌트
│   ├── blank.vue             # 빈 레이아웃 (로그인 등)
│   ├── default.vue           # 기본 레이아웃
│   ├── popBlank.vue          # 팝업 레이아웃
│   └── components/           # 레이아웃 하위 컴포넌트
│       ├── DefaultLayoutWithVerticalNav.vue
│       ├── Footer.vue
│       ├── NavbarThemeSwitcher.vue
│       ├── NavItems.vue
│       └── UserProfile.vue
└── UpgradeToPro.vue         # 업그레이드 컴포넌트

버튼 컴포넌트 ​

ButtonBox.vue ​

공통 버튼 컴포넌트입니다.

사용 예제:

vue
<template>
  <ButtonBox
    text="저장"
    color="primary"
    @click="handleSave"
  />
</template>

입력 컴포넌트 ​

InputPhone.vue ​

전화번호 입력 컴포넌트입니다.

주요 기능:

  • 전화번호 형식 자동 포맷팅
  • 유효성 검증

사용 예제:

vue
<template>
  <InputPhone
    v-model="phoneNumber"
    placeholder="010-1234-5678"
  />
</template>

로딩 컴포넌트 ​

Loading.vue ​

전역 로딩 인디케이터 컴포넌트입니다.

사용 방법: 로딩 스토어를 통해 자동으로 표시/숨김 처리됩니다.

javascript
import { useLoadingStore } from '@/utils/stores/loadingStore.js'

const loadingStore = useLoadingStore()
loadingStore.showLoading()  // 로딩 표시
loadingStore.hideLoading()  // 로딩 숨김

조회 컴포넌트 (lookup) ​

LookUpBox.vue ​

데이터 조회 및 선택 컴포넌트입니다.

주요 기능:

  • 팝업을 통한 데이터 조회
  • 다중 선택 지원
  • 검색 기능

사용 예제:

vue
<template>
  <LookUpBox
    v-model="selectedData"
    :url="'/api/direct/call/atomic'"
    :procedure="'P_LOOKUP_Q'"
    @select="handleSelect"
  />
</template>

선택 컴포넌트 (select) ​

SelectBoxSQL.vue ​

SQL 기반 선택 박스 컴포넌트입니다.

주요 기능:

  • 비즈니스 컴포넌트 ID를 통한 데이터 조회
  • 동적 옵션 로딩

사용 예제:

vue
<template>
  <SelectBoxSQL
    v-model="selectedValue"
    v-bind="jsonData"
  />
</template>

<script setup>
const jsonData = reactive({
  bizcompId: 'BIZCOMP_ID',
  whereClause: '',
  compCode: '1000',
})
</script>

SelectBoxDate.vue ​

날짜 선택 컴포넌트입니다.

주요 기능:

  • 날짜 범위 선택
  • 날짜 형식 지정

사용 예제:

vue
<template>
  <SelectBoxDate
    v-model="selectedDate"
    format="YYYY-MM-DD"
  />
</template>

MultiSelectBoxSQL.vue ​

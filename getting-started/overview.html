프로젝트 개요 ​

프로젝트 소개 ​

AXP (AXP Framework)는 Vue 3 기반의 엔터프라이즈급 프론트엔드 애플리케이션 프레임워크입니다. 이 프로젝트는 다양한 비즈니스 모듈(ESS, FCM, geosan 등)을 지원하는 통합 관리 시스템의 프론트엔드 부분을 담당합니다.

주요 특징 ​

  • Vue 3 Composition API: 최신 Vue 3의 Composition API를 활용한 컴포넌트 기반 개발
  • Vite 빌드 도구: 빠른 개발 서버와 최적화된 프로덕션 빌드
  • Pinia 상태 관리: Vuex의 후속작인 Pinia를 사용한 중앙 집중식 상태 관리
  • Vuetify UI 프레임워크: Material Design 기반의 풍부한 UI 컴포넌트 라이브러리
  • 모듈화된 구조: ESS, FCM, geosan 등 다양한 비즈니스 모듈을 독립적으로 관리
  • JWT 기반 인증: 안전한 토큰 기반 인증 시스템
  • 자동 컴포넌트/함수 임포트: unplugin-auto-import와 unplugin-vue-components를 통한 자동 임포트

프로젝트 버전 ​

  • 프로젝트명: AXP
  • 버전: 2.3.0
  • Vue 버전: 3.5.13
  • Vite 버전: 5.4.11

기술 스택 요약 ​

핵심 프레임워크 ​

  • Vue 3.5.13 (Composition API)
  • Vite 5.4.11
  • Pinia 2.3.0
  • Vue Router 4.5.0

UI 프레임워크 ​

  • Vuetify 3.7.5
  • Material Design Icons

주요 라이브러리 ​

  • Axios 1.9.0 (HTTP 클라이언트)
  • AG Grid 33.3.2 (데이터 그리드)
  • ApexCharts 3.54.1 (차트)
  • ECharts 5.6.0 (차트)
  • ExcelJS 4.4.0 (Excel 처리)
  • CryptoJS 4.2.0 (암호화)

프로젝트 목적 ​

이 프로젝트는 다음과 같은 목적으로 개발되었습니다:

  1. 통합 관리 시스템: 다양한 비즈니스 모듈을 하나의 플랫폼에서 관리
  2. 재사용 가능한 컴포넌트: 공통 UI 컴포넌트를 통한 개발 효율성 향상
  3. 확장 가능한 아키텍처: 새로운 모듈과 기능을 쉽게 추가할 수 있는 구조
  4. 개발자 경험 향상: 자동 임포트, 타입 추론 등을 통한 개발 편의성 제공

주요 모듈 ​

AXP Framework ​

프로젝트의 핵심 프레임워크 모듈로, 인증, 메인 대시보드 등의 기본 기능을 제공합니다.

ESS 모듈 ​

Employee Self Service 모듈로, 인사 관리, 교육, 신용카드 관리 등의 기능을 포함합니다.

FCM 모듈 ​

Finance & HR 모듈로, ERP 및 HR 관련 기능을 제공합니다.

geosan 모듈 ​

BI(Business Intelligence) 관련 기능을 제공하는 모듈입니다.

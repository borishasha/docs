SMS 발송 API ​

SMS 메시지를 발송하는 API입니다.

기본 정보 ​

  • Base URL: /api/admin
  • HTTP Method: POST
  • 인증: JWT 토큰 필요

API 목록 ​

1. SMS 메시지 발송 ​

엔드포인트: /api/admin/AsSMSManagement/msgSend

설명: SMS 메시지를 발송합니다. 즉시 발송 또는 예약 발송을 지원합니다.

요청 본문:

json
{
  "params": {
    "msg_type": "SMS",
    "rev_type": "10",
    "rev_type_value": "",
    "msg_cont": "메시지 내용",
    "send_tel": "01012345678",
    "dt_dtl_no": "1",
    "dt_send_name": "홍길동",
    "dt_comy": "회사명",
    "dt_depart": "부서명",
    "dt_posi": "직급",
    "dt_mo_tel": "01012345678",
    "dt_email": "user@example.com"
  },
  "sessionInfo": {
    "CompCode": "COMP001",
    "LangID": "KOR",
    "UserID": "USER001",
    "ipAddress": "192.168.1.100"
  }
}

파라미터:

params (필수) ​

  • msg_type (String, 필수): 메시지 타입
    • "SMS": SMS
    • "LMS": LMS
  • rev_type (String, 필수): 발송 타입
    • "10": 즉시 발송
    • "20": N분 후 발송
    • "30": N시간 후 발송
    • "40": 특정 일시 발송
  • rev_type_value (String, 선택): 발송 시간 값
    • rev_type="20": 분 수 (예: "5")
    • rev_type="30": 시간 수 (예: "2")
    • rev_type="40": 일시 (예: "20240101120000" 또는 "20240101")
  • msg_cont (String, 필수): 메시지 내용
  • send_tel (String, 필수): 발신 전화번호
  • dt_dtl_no (String, 필수): 상세 번호
  • dt_send_name (String, 필수): 발신자 이름
  • dt_comy (String, 선택): 회사명
  • dt_depart (String, 선택): 부서명
  • dt_posi (String, 선택): 직급
  • dt_mo_tel (String, 필수): 수신 전화번호
  • dt_email (String, 선택): 이메일

sessionInfo (필수) ​

  • CompCode (String, 필수): 회사 코드
  • LangID (String, 필수): 언어 ID
  • UserID (String, 필수): 사용자 ID
  • ipAddress (String, 필수): IP 주소

응답 예시 (성공):

json
{
  "status": "S",
  "message": "",
  "data": {
    "CV_1": [
      {
        "mst_no": "MSG001"
      }
    ],
    "V_P_ERROR_CODE": "MSG0001",
    "isok": "Y"
  }
}

응답 필드:

  • CV_1[0].mst_no (String): 발송 메시지 번호 (즉시 발송인 경우)
  • V_P_ERROR_CODE (String): 에러 코드 ("MSG0001" = 성공)
  • isok (String): 최종 처리 결과 ("Y" = 성공, "N" = 실패)

응답 예시 (실패):

json
{
  "status": "E",
  "message": "SMS 전송에 실패했습니다.",
  "data": {
    "isok": "N",
    "V_P_ERROR_CODE": "ERROR001"
  }
}

발송 타입별 동작:

  1. 즉시 발송 (rev_type="10")

    • DB 저장 후 즉시 SMS 발송
    • 발송 결과를 응답에 포함

  2. 예약 발송 (rev_type="20", "30", "40")

    • DB에 저장만 하고 발송하지 않음
    • 스케줄러가 예약 시간에 발송

사용 예시:

javascript
// 즉시 발송
const response = await axios.post('/api/admin/AsSMSManagement/msgSend', {
  params: {
    msg_type: 'SMS',
    rev_type: '10',
    rev_type_value: '',
    msg_cont: '안녕하세요. 테스트 메시지입니다.',
    send_tel: '01012345678',
    dt_dtl_no: '1',
    dt_send_name: '홍길동',
    dt_mo_tel: '01087654321'
  },
  sessionInfo: {
    CompCode: 'COMP001',
    LangID: 'KOR',
    UserID: 'USER001',
    ipAddress: '192.168.1.100'
  }
});

if (response.data.status === 'S' && response.data.data.isok === 'Y') {
  console.log('SMS 발송 성공');
} else {
  console.log('SMS 발송 실패');
}

메시지 상태 ​

  • 10: 발송 대기
  • 20: 발송 완료
  • 30: 발송 실패

주의사항 ​

  1. 즉시 발송: rev_type="10"인 경우에만 즉시 발송됩니다.
  2. 예약 발송: 스케줄러가 실행되어야 발송됩니다.
  3. 메시지 내용: 특수 문자 제한이 있을 수 있습니다.
  4. 전화번호 형식: 하이픈(-) 없이 숫자만 입력 (예: "01012345678")

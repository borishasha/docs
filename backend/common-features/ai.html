AI 서비스 ​

AxpFramework는 Google Gemini AI를 연동합니다.

설정 ​

ai-config.yaml ​

yaml
gemini:
  api-key: your-api-key

사용 예시 ​

java
@Autowired
private AxpGeminiService geminiService;

Map<String, Object> request = new HashMap<>();
request.put("prompt", "질문 내용");

String response = geminiService.callGeminiAi(request);

API 엔드포인트 ​

POST /api/ai/genai

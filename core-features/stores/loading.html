로딩 스토어 (loadingStore) ​

개요 ​

loadingStore는 애플리케이션의 로딩 상태를 관리하는 Pinia 스토어입니다.

파일 위치 ​

src/utils/stores/loadingStore.js

상태 (State) ​

javascript
state: () => ({
  isLoading: false,
})

액션 (Actions) ​

showLoading() ​

로딩 상태를 활성화합니다.

hideLoading() ​

로딩 상태를 비활성화합니다.

사용 예제 ​

vue
<template>
  <Loading :isLoading="loadingStore.isLoading" />
</template>

<script setup>
import { useLoadingStore } from '@/utils/stores/loadingStore.js'

const loadingStore = useLoadingStore()

const fetchData = async () => {
  loadingStore.showLoading()
  try {
    // API 호출
  } finally {
    loadingStore.hideLoading()
  }
}
</script>

주의사항 ​

CustomAxios.js의 인터셉터에서 자동으로 로딩 상태가 관리되므로, 대부분의 경우 수동으로 호출할 필요가 없습니다.
